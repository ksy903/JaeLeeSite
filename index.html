High School

Date: 2014 - 2015

I first started programming during High School Junior year learning basic programming concepts using Java. I became very passionate about programming and quickly reached out for more oppotunities to be involved in programming. That led to my first programming competition. I really enjoyed the experience, so my friend and I formed the Programming Club at my high school, where every day after class, we would lead the study sessions and drill practice problems. The club would often get new members that I often helped teach, dual programming with them with various programming problems and explaining java specifics and abstract concepts tested for the written portion of the programming competitions. Gradually my personal skills increased, and hit first place team and first place written portion placement at the district level. While I did not place in the regional competition at Baylor University, the extracurricular activity helped me build a solid foundation in programming through repetitive drills and group study sessions. It also gave me a solid enough foundation to get certified as an Oracle Certified Assossiate in Java Standard Edition 7.
